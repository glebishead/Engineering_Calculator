Техническое задание на разработку инженерного калькулятора

Необходимо разработать программу, которая позволит использовать функции математической физики, 
тригонометрические функции, обратные тригонометрические и т.п. 
Функции линейной алгебры для нахождения корней уравнения. 
Хранения различных физических и математических констант, 
таких как ускорение свободного падения, гравитационная постоянная, скорость света и т.д. 

Программа будет представлять из себя образ калькулятора с расширенным функционалом. 
Все результаты вычислений будут сохраняться в базе данных для экономии времени при проведении подсчётов и возможности их анализа.

Структура базы данных:
здесб картинка бд


Процесс взаимодействия с пользователем будет реализован с помощью окна программы. 
Оно будет способно открывать другие окна, в которых будет разниться функционал. Так, к примеру, в окне
обычного калькулятора будут реализованы простейшие математические операции. В инженерном добавятся постоянные,
тригонометрические и обратные тригонометрические функции. В ином окне будут храниться некоторые физические постоянные:
ускорение свободного падения, гравитационная потоянная, скорость света и т.п. Последним же окном будет изображение графика
некой заданной функции, от прямой, параболы, до более сложных.


Задачи для разработки:
Несколько файлов frontend-составляюшей проекта, на каждый из калькуляторов
Несколько файлов .py с классами, реализующими алгоритмы работы калькуляторов
Механизм заполнения базы данных результатами выполнения вычислений
